# AI TRPG（仮題）

**「ルールはコードが守り、物語はAIが紡ぐ」1人用TRPG。**

プレイヤーは自由なテキスト入力で行動を宣言し、AIのゲームマスターが物語を描写する。判定・戦闘・リソース管理などのゲームルールはすべてコード側で決定論的に処理し、AIは「確定した結果の描写」だけを担当する。ローカルLLMを同梱し、完全オフライン・買い切り・追加課金なしでSteamリリースを目指す。

## 特徴

- 🎲 **公平なダイス** — 乱数・判定・状態変更はすべてコード。AIはズルも忖度もできない
- 📖 **自由なロールプレイ** — 行動宣言は自由入力。2d6の5段階判定（部分成功あり）が物語を停滞させない
- 🔌 **完全オフライン** — llama.cpp + Qwen3（8B/4B自動選択）を同梱。外部AI接続は明示的なオプトイン
- 💰 **追加課金なし** — 生成回数への従量課金（インク方式）は導入しない

## 技術スタック

Godot 4 / GDScript / llama.cpp（GDExtension）/ Qwen3 GGUF（Apache 2.0）

## 核となる設計原則（抜粋）

- AIはゲーム状態を変更しない。状態変更はすべてコードが行う（INV-1）
- AIの出力は文法制約デコード＋コード検証を通してのみゲームロジックに入る（INV-3）
- 更新順序は常に「判定確定 → 状態更新 → 保存 → AI描写生成」。生成失敗でダイス結果は失われない（INV-6）
- デフォルト完全オフライン。テレメトリの無断送信は永久に行わない（INV-4）
